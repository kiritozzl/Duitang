# Duitang  
ViewPager +StaggeredGridView      以及    ViewPager + StaggeredGridLayoutManager 实现瀑布流显示图片的效果
Toolbar的基本使用
使用的开源库：
*StaggeredGridView
*SwipeRefreshLayout
*pagerslidingtabstrip
*butterknife
*picasso
*circleimageview
学习的来源：
https://github.com/KayWu/Duitang
